# Curt Vance Portfolio

Modern frontend portfolio built with:

* Next.js
* React
* TypeScript
* Tailwind CSS
* EmailJS

## Features

* Responsive design
* Animated project cards
* Skeleton loading states
* Contact form integration
* Modern UI/UX design
* Mobile optimized
