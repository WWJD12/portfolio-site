# FES GitHub Copilot Course - Practice Repository

Welcome to the FES GitHub Copilot Course practice repository! This workspace contains all the files you'll need to follow along with the course.

## 🚀 Getting Started

1. **Install Prerequisites:**

   - Visual Studio Code (latest version)
   - GitHub Copilot extension
   - GitHub Copilot Chat extension
   - Node.js (v18 or higher)

2. **Sign in to GitHub Copilot:**

   - Open VS Code
   - Sign in with your GitHub account when prompted
   - Verify the Copilot icon in the bottom-right status bar

3. **Install Dependencies:**

   ```bash
   npm install
   ```

4. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎯 How to Use This Repo

Each module folder contains practice files with **BOLD COMMENTS** marking exactly where you should:

- ✍️ **DO EXERCISES HERE**
- 🔧 **FIX CODE HERE**
- ➕ **ADD COMPONENTS HERE**
- 🎨 **REFACTOR THIS**

Follow the course lessons and look for these markers to know where to apply what you're learning!

## 🔑 Key Shortcuts

- **Tab** → Accept suggestion
- **Esc** → Dismiss suggestion
- **Ctrl/Cmd + I** → Inline Chat (inside file)
- **Ctrl/Cmd + Shift + I** → Quick Chat (floating box)
- **Alt/Option + ]** → Next suggestion
- **Alt/Option + [** → Previous suggestion

## 💡 Tips

- Keep this workspace open in VS Code while following the course
- Use Copilot as a coding partner, not just autocomplete
- Iterate and refine - the first suggestion isn't always the best
- Review Copilot's code before accepting it
